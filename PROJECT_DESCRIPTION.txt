================================================================================
VANGUARD - Fleet Control Tower
Fleet Monitoring & GPS Tracking Application
================================================================================

DESKRIPSI PROJECT
--------------------------------------------------------------------------------

VANGUARD adalah aplikasi web fleet monitoring dan GPS tracking yang dirancang untuk
memonitor dan mengelola armada kendaraan operasional secara realtime. Aplikasi ini
terinspirasi dari TRAMOS Systems dan dibangun dengan teknologi modern untuk
menyediakan pengalaman yang lebih clean, professional, dan user-friendly.

Aplikasi ini berfungsi sebagai "control tower" bagi dispatcher, supervisor,
admin transport, dan tim operasional untuk:


  - Memantau posisi kendaraan secara realtime
  - Mengelola task pengiriman dan assignment kendaraan
  - Melihat route yang sudah dilalui dan route yang direncanakan
  - Menerima notifikasi event seperti speeding, geofence, GPS delayed
  - Membuat laporan operasional
  - Melihat bukti kamera dari camera snapshot dan dashcam
  - Mengelola data kendaraan, driver, dan user


================================================================================
FITUR UTAMA
================================================================================

1. MONITORING
-------------

   1.1 Dashboard
       - Ringkasan status armada (total, driving, idle, parking, offline)
       - Vehicle status summary
       - Attention alerts (GPS delayed, low fuel, overspeed)
       - Quick access ke modul penting
       - Status distribution dengan visualisasi progress bar

   1.2 Realtime Monitor
       - Peta realtime dengan marker kendaraan
       - Table view untuk scanning cepat
       - Filter berdasarkan status kendaraan
       - Search unit atau driver
       - Detail panel untuk kendaraan terpilih
       - Layer control (track, geofence, checkpoint)
       - WebSocket support untuk live updates

   1.3 Task Monitor (TMS)
       - Table view dengan kolom lengkap (vehicle, driver, group, time, status, task, origin, destination)
       - Map view dengan route visualization
       - Status task: Progress, Unloading, Waiting, Assigned
       - Filter dan search
       - Sortable columns
       - Detail panel dengan informasi trip lengkap
       - Double-click untuk buka vehicle di map
       - Toolbar: Table/Map toggle, Refresh, Order By, Manage, Customize Column

   1.4 Locate
       - Pencarian unit cepat
       - Recent searches
       - Detail posisi terakhir kendaraan
       - Copy coordinates
       - Navigasi ke Realtime Monitor dan History

   1.5 Geofence
       - Daftar zone dengan filter type (Depot, Customer, Port, Checkpoint)
       - Map visualization dengan circle markers
       - Toggle visibility per zone
       - Detail panel dengan koordinat
       - Add, Edit, Delete geofence


2. OPERATION
-----------

   2.1 Vehicle Registry
       - Master data kendaraan
       - Informasi GPS device (ID, SIM)
       - Ownership (Company/Vendor)
       - KIR due date tracking
       - Service due monitoring
       - Odometer tracking
       - Device health status
       - Sortable dan filterable table
       - Export functionality

   2.2 Driver Registry
       - Master data supir
       - Safety score tracking
       - License/SIM management
       - RFID pairing
       - Trip count dan total KM
       - Incident history
       - Detail panel dengan profile lengkap
       - Sortable columns

   2.3 History
       - Trip replay dan investigasi
       - Timeline view dengan visualisasi route
       - Detail view dalam format table
       - Tab categories: Timeline, Detail, Engine, Driving, Idle, Stop, Speeding, Events, Reverse, Geofence
       - Playback controls (1x, 2x, 4x speed)
       - Date range selector
       - Unit/Driver filter
       - Export CSV

   2.4 Accident Log
       - Catatan insiden dan safety events
       - Severity classification (High, Medium, Low)
       - Status tracking (Review, Action, Closed)
       - Evidence management
       - Filter by severity dan status
       - Review evidence functionality
       - Mark as resolved


3. EVIDENCE
-----------

   3.1 Report Center
       - 13 jenis report:
         * Trip Summary
         * Idle Report
         * Stop Report
         * Working Time
         * Reverse Summary
         * DIMS (Driver Improvement)
         * Monthly Summary
         * Fatigue Summary
         * Mileage Day
         * Alarm Video
         * Speeding
         * Fuel Usage
         * Geofence Events
       - Report generator dengan preview
       - Export PDF/CSV
       - Report queue
       - Supervisor review stats

   3.2 Camera Snapshot
       - Gallery dengan thumbnail
       - Event types: Arrival proof, Speed alert, Fuel stop, Engine off, Customer gate, Geofence entry
       - Filter by event type
       - Search unit/location
       - Review modal dengan detail lengkap
       - Download full resolution

   3.3 Dashcam Monitor
       - Video player interface
       - Multi-channel support (Front, Cabin, Rear)
       - Status: Live, Review, Offline
       - Playback controls
       - Play/Pause, Skip, Mute
       - Snapshot capture
       - Fullscreen mode
       - Channel selector
       - Recent events list


4. ADMINISTRATION
----------------

   4.1 Control Panel
       - Units Configuration
       - Drivers Configuration
       - Users & Roles Management
       - Telegram Alerts Configuration
       - API Webhook Settings
       - Database Retention Policy
       - Smart handling rules display

   4.2 Settings
       - Theme preference (Light/Dark/System)
       - Table density (Compact/Comfortable)
       - Map preferences (Tile provider, Zoom level)
       - Notification settings (Speeding, Geofence, Offline, Idle, Fuel)
       - Sound alerts toggle
       - Auto-refresh interval
       - Account information
       - Session status


================================================================================
TEKNOLOGI STACK
================================================================================

FRONTEND
--------
  - Framework: Next.js 14 (App Router)
  - Language: TypeScript
  - Styling: Tailwind CSS
  - UI Components: Custom components dengan Lucide React icons
  - Maps: Leaflet dengan React-Leaflet
  - State Management: React hooks (useState, useMemo, useEffect)
  - Dark Mode: CSS custom properties + Tailwind dark mode

BACKEND
--------
  - Framework: FastAPI (Python)
  - Database: SQLite (development), PostgreSQL (production ready)
  - WebSocket: Native FastAPI WebSocket
  - API: RESTful endpoints
  - ORM: SQLAlchemy

INFRASTRUCTURE
--------------
  - Container: Docker & Docker Compose
  - Services: FastAPI, PostgreSQL, Redis (optional cache)

================================================================================
CARA MENJALANKAN
================================================================================

MENGGUNAKAN DOCKER (Recommended)
--------------------------------
1. Pastikan Docker dan Docker Compose terinstall
2. Clone repository
3. Jalankan:
   docker-compose up

   Atau untuk development:
   docker-compose up --build

   Frontend: http://localhost:3000
   Backend API: http://localhost:8000
   API Docs: http://localhost:8000/docs


MENGGUNAKAN LOCAL DEVELOPMENT
-----------------------------
1. Clone repository
2. Setup frontend:
   cd frontend
   npm install
   npm run dev

3. Setup backend (optional untuk mock data):
   cd backend
   python -m venv venv
   source venv/bin/activate  # Windows: venv\Scripts\activate
   pip install -r requirements.txt
   python -m uvicorn main:app --reload

4. Buka browser:
   http://localhost:3000
